Upload Our Local System Files To GitHub
1)	Open cmd to related folder
2)	git init
3)	git status
4)	git add .
5)	git commit –m “Type Any Comment Related To Files”
6)	git remote
7)	git remote origin master (Past The GitHub Repository Clone and Download Copy Link)
8)	git remote add origin (Past The GitHub Repository Clone and Download Copy Link)
9)	git remote –v
10)	git fetch origin
11)	git push –f origin master
